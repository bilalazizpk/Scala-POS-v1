# Scala POS Frontend

React + Vite + Tailwind CSS + shadcn components

## Development

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Product Management with Search
- Order Management with Sales Tracking
- Real-time Data Binding
- RESTful API Integration with .NET Backend
- Responsive Design with Tailwind CSS
- Modern Component Library with shadcn
